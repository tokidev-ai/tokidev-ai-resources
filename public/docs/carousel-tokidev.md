<h2 id="que-es">Qué Genera</h2>

Le decís el tema, Claude genera el carrusel completo en HTML — listo para abrir en el navegador y exportar cada slide como PNG en resolución exacta **1080×1350px**.

No necesitás explicarle colores, tipografía ni estructura. El sistema de diseño está incluido en la skill — listo para usar.

<h2 id="instalacion">Instalación</h2>


<div class="doc-steps">
  <div class="doc-step">
    <span class="doc-step-num">01</span>
    <div class="doc-step-body">
      <strong>Descargá el SKILL.md</strong>
      <p>Hacé clic en el botón de arriba para abrir el archivo en GitHub. Guardalo con Ctrl+S.</p>
    </div>
  </div>
  <div class="doc-step">
    <span class="doc-step-num">02</span>
    <div class="doc-step-body">
      <strong>Abrí Claude Cowork o Claude Code</strong>
      <p>La skill guarda los archivos HTML generados en una carpeta de trabajo local. Necesitás usarla en un Project conectado a una carpeta.</p>
    </div>
  </div>
  <div class="doc-step">
    <span class="doc-step-num">03</span>
    <div class="doc-step-body">
      <strong>Subí el archivo al Project</strong>
      <p>Arrastrá el SKILL.md a las instrucciones del Project. Claude lo lee en cada conversación automáticamente.</p>
    </div>
  </div>
  <div class="doc-step">
    <span class="doc-step-num">04</span>
    <div class="doc-step-body">
      <strong>Exportá los PNG</strong>
      <p>Abrí el HTML generado en el navegador y hacé clic en "⬇ Exportar N slides PNG". Se descargan todos, listos para subir a Instagram.</p>
    </div>
  </div>
</div>

<h2 id="como-usarla">Cómo Usarla</h2>

Escribile a Claude en ese Project:

```
Hacé un carrusel sobre [TEMA]
```

```
Nuevo carrusel de 8 slides sobre las mejores herramientas de IA para developers
```

```
Carrusel de 10 slides sobre cómo optimizar tokens en Claude
```

Claude pregunta cuántos slides querés antes de generar (mínimo 3, por defecto 7). Si ya lo especificás en el mensaje, arranca directo.

<div class="doc-note">
  <span class="material-icons">lightbulb</span>
  <span>El slide de portada y el CTA final son siempre fijos. Los slides del medio se adaptan a la cantidad que pidas.</span>
</div>
